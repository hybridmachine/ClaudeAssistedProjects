# Pattern library attribution

The RLE pattern files in this directory are public-domain patterns from the
original 1969–1971 publications of Conway's Game of Life (primarily Martin
Gardner's "Mathematical Games" column in *Scientific American*, October 1970),
or are trivial patterns whose discovery predates meaningful copyright
protection for short configurations of live cells.

Per the licensing decision recorded in `PLAN-pattern-library.md` §1, this
initial bundled library ships **only** patterns that are public-domain or
ineligible for copyright. Patterns sourced from CC-BY-SA repositories like
LifeWiki have been deliberately excluded until the repository's overall
license is reviewed for CC-BY-SA compatibility.

## Adding new patterns

When adding new patterns, include provenance in the file's `#C` comment lines
(e.g. discoverer, year, source URL) and update the table above. If a pattern
is licensed under CC-BY-SA or any other non-public-domain license, coordinate
with the repository maintainer first — the project has not yet adopted a
CC-BY-SA-compatible license.
